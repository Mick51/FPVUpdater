FPV Updater
FPV Updater est une application Android moderne conçue pour les pilotes de FPV (First Person View). Elle permet de suivre en temps réel les dernières versions des firmwares et logiciels essentiels à la pratique du FPV.

🚀 Fonctionnalités
Vue d'ensemble centralisée : Accès rapide aux dernières releases de vos projets FPV préférés (Betaflight, INAV, EdgeTX, ExpressLRS, etc.).

Mise à jour en temps réel : Récupération automatique des données via l'API GitHub.

Interface intuitive : Design moderne basé sur Material Design 3 avec support du mode sombre.

Accès rapide : Liens directs vers les dépôts officiels et les pages de téléchargement.

📱 Aperçu technique
Langage : Kotlin

Framework UI : Jetpack Compose

Architecture : MVVM (Model-View-ViewModel)

Networking : Retrofit & OkHttp

Image Loading : Coil

Design System : Material 3

📦 Installation
Clonez ce dépôt sur votre machine.

Ajoutez votre GITHUB_TOKEN dans le fichier local.properties à la racine du projet :

Properties
GITHUB_TOKEN=votre_token_ici
Ouvrez le projet dans Android Studio et lancez la compilation.

🛡️ Politique de confidentialité
Cette application ne collecte aucune donnée personnelle. Elle se contente de lire des informations publiques via l'API GitHub pour afficher les versions des logiciels.
